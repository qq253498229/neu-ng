# neutralinojs-angular-template

个人使用的Neutralinojs脚手架，前端使用angular。

添加了一些常用软件，比如lodash、zorro、ngxs等。

## 如何使用

使用该模板创建应用，其中 `myapp` 是你的项目文件夹名称：

```bash
neu create --template qq253498229/neu-ng myapp
cd myapp
```

## 如何启动开发服务器

```bash
neu run
```

## 如何打包

```bash
neu build --release
```
